# FastAPI-MiniProject


For this project ->

1 - create venv 
``` text
    python -m venv .venv \
```
or 
``` text
  python3 -m venv .venv
```
2 - activate <br>
---2.1 - Windows 
  ``` text
  .\.venv\Scripts\activate
```
---2.2 - linux \ macos
   ``` text
  source .venv\bin\activate
```
3 - installing  libs
  ``` text
  pip install -r requirements.txt
```
4 - Create .env file and paste your SECRET_KEY 

5 - run in run.py

6 - go to

``` text
  http://127.0.0.1:8000/docs
```
its fastapi Swagger UI 

7 - Enjoy
